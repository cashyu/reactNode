# react+express构建博客系统
<h3>描述</h3>
<br/>
<p>兼顾见后端练手项目，适合初学者</p>
<h3>主要技术栈</h3>
<br />
<ul>
  <li>nodejs</li>
  <li>react</li>
  <li>redux</li>
  <li>react-router</li>
  <li>express</li>
  <li>mongodb</li>
</ul>
<h3>运行</h3>
<br>
<p>1. git clone https://github.com/cashyu/reactNode.git</p>
<p>2. cd reactNode/client/</p>
<p>3. npm run build</p>
<p>4. cd ../server</p>
<p>5. node inde.js</p>
<p>6. 打开浏览器，访问http://localhost:3000</p>
